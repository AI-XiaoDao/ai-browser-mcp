# 火山编译生成的 C++ 源码（Release x64 / win32）

本目录 = 本地编译 `_int/.../project/` 的 **Git 同步副本**，由火山 IDE 将 [`../src/`](../src/) 中的 `.wsv` **自动翻译**而来，供审计与对照。

## 在「四层模型」中的位置

| 层级 | 本目录 | 本地 `_int` 对应 | 说明 |
|:--:|--------|------------------|------|
| ① | — | — | 权威源码在 **`../src/*.wsv`**，改代码请去那里 |
| **②** | **`release-x64/`、`release-win32/`** | **`project/`** | 生成 C++，**勿手工编辑** |
| ③ | — | `linker/out/` | `.obj`/`.pch`，**不是源码**，勿打包 |
| ④ | — | `linker/`（除 `out/`） | exe + CEF，走 GitHub Releases |

完整对照表见仓库 [README · 开源范围与火山编译目录](../../../README.md#-开源范围与火山编译目录)。

## 目录内容

| 文件 | 说明 |
|------|------|
| `vpkg_*.cpp` | 各 `.wsv` 翻译单元（如 `vpkg_MCP_Server.cpp`） |
| `vcls_*.h` | 火山类/类型头文件 |
| `stdafx.cpp` / `stdafx.h` | 预编译头 |
| `makefile` | nmake 脚本（`OUT_DIR=../linker/out`） |
| `AI浏览器.rc` | 资源脚本 |

文件头均含 `Generated by Volcano Development Platform`。

## 同步与发版

| 项 | 值 |
|----|-----|
| 版本 | 与 `MCP_版本号` 一致（当前 2.8.0） |
| x64 | 编译 Release x64 后，复制 `project/` → `release-x64/` |
| win32 | 编译 Release win32 后，复制 `project/` → `release-win32/` |
| 自动 | `.\release\pack-release.ps1 -Version 2.8.0 -Platform all` |

## 常见误区

- **`linker/out/` 不是 C++ 源码** — 只是链接前的目标文件
- **不要直接改本目录提 PR** — 改 `src/*.wsv`，重新编译后再同步
