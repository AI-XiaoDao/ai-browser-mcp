## w_ado

**包**：`火山.视窗.数据库` | **基于ADO实现对各种不同数据库的访问**

### 数据库类

**连接方法**：
- `连接Mysql(地址, 数据库名, 用户名, 密码, 端口号=3306, 驱动程序="MySQL ODBC 8.0 Unicode Driver", 异步连接=假)` → 逻辑型
- `连接Mssql(地址, 数据库名, 用户名, 密码, 异步连接=假, 加密连接=假)` → 逻辑型
- `连接Access(文件路径, 密码="", 是否新版=假)` → 逻辑型
- `连接Postgresql(地址, 数据库名, 用户名, 密码, 端口号=5432, 驱动程序="", 异步连接=假)` → 逻辑型
- `连接Oracle(地址, 用户名, 密码, 驱动程序="Oracle in OraClient11g_home1", 异步连接=假)` → 逻辑型
- `连接(连接文本="", 用户名="", 密码="", 是否客户端=真, 异步连接=假)` → 逻辑型 | 通用连接

**属性**（读写）：`连接文本`, `执行超时`(秒), `连接超时`(秒), `当前状态`, `提供者名称`, `版本信息`(只读), `游标服务位置`, `默认数据库`, `事务隔离级别`, `权限模式`

**操作**：
- `执行SQL(语句内容, 选项=-1, 影响行数)` → 记录集类 | ⭐返回只读只进记录集
- `取结构信息(查询类型)` → 记录集类
- `关闭()`, `取消执行()`
- `开始事务()` → 整数, `保存事务()`, `回滚事务()`
- `取错误信息()` → 数据库错误集合类

**事件**：`即将开始连接`, `即将开始执行`, `警告信息反馈`, `开始事务完成`, `提交事务完成`, `回滚事务完成`, `命令执行完成`, `连接完成`, `断开连接`

### 记录集类（基于COM对象类）

**属性**（读写）：`当前页`, `当前位置`, `首记录前`(BOF), `尾记录后`(EOF), `缓存数量`, `游标服务位置`, `游标类型`, `编辑状态`, `书签`, `筛选器`, `索引名称`, `编辑锁类型`, `数据封送选项`, `最大记录数`, `页数`(只读), `单页记录数`, `记录数量`(只读), `排序规则`, `数据源`, `状态`, `记录状态`, `保持同步`

**导航**：`到首记录()`, `到尾记录()`, `到下一行()`, `到上一行()`, `到下一记录集()`, `移动位置(行数, 起始移动书签=空对象)`

**CRUD**：
- `添加空记录()`, `添加新记录(字段列表, 值列表)`
- `更新(字段=空对象, 值=空对象)`, `批更新(范围选项=所有)`
- `删除记录(选项=当前)`, `取消更新()`, `取消批更新(选项)`
- `查找指定记录(条件语句, 起始搜索位置=0, 正向搜索=真, 起始搜索书签=空对象)`

**数据读写**：
- `读数据(字段名)` → COM变体型, `读索引数据(字段索引)` → COM变体型
- `读文本数据/读整数数据/读长整数数据/读小数数据/读字节集数据(字段名)`
- `读索引文本数据/读索引整数数据/读索引长整数数据/...(字段索引)`
- `写数据(字段名, 值)`, `写文本数据/写整数数据/...(字段名, 值)`
- `写索引数据/写索引文本数据/...(字段索引, 值)`

**序列化**：`到文本(行数=-1, 列分隔符="", 行分隔符="", 空占位符="")` → 文本型, `到字节集()` → 字节集类, `置字节集(记录集数据)`, `保存到文件(文件, 类型=ADTG)`
**其他**：`取多行记录(行数=-1, 起始位置, 字段列表)` → COM变体型(二维数组), `取字段列表()` → 数据库字段集合类, `刷新记录()`, `重新同步()`, `打开()`, `从已有连接打开()`, `是否支持指定功能(功能选项)`

## w_json

**包**：`火山视窗.JSON操作` | **基于jsoncpp-1.9.7**（另一套JSON封装）

- `到可读文本(是否格式化=假, 是否不转义=真)` → 文本型

- `创建自文本(内容)` → 逻辑型, `创建自字节集(数据)` → 逻辑型
- `清空()`, `是否为空()`, `成员数`(属性读)
- `启用顺序输出()`(静态), `启用线程安全()`(静态)

### JSON对象类（基础类=JSON基础类）

**取值**（静态方法，第一参数为对象实例）：
- `取文本值(对象, 键名)`, `取整数值(对象, 键名)`, `取长整数值(对象, 键名)`, `取小数值(对象, 键名)`, `取逻辑值(对象, 键名)`
- `取对象值(对象, 键名, 结果)`, `取数组值(对象, 键名, 结果)`
- `取路径文本值/整数值/...(对象, 路径)` — 支持 `key1.key2.arr[0]` 路径

**置值**（静态）：
- `置入文本值(对象, 键名, 值)`, `置入整数值(对象, 键名, 值)`, `置入长整数值(...)`, `置入小数值(...)`, `置入逻辑值(...)`
- `置入对象值(对象, 键名, 值对象)`, `置入数组值(对象, 键名, 值数组)`
- `置入路径文本值/整数值/...(对象, 路径, 值)`

**其他**：`是否存在(对象, 键名)`, `路径是否存在(对象, 路径)`, `删除成员(对象, 键名)`, `取类型(对象, 键名)` → JSON类型

### JSON数组类（基础类=JSON基础类）

**取值**（静态）：`取文本值(数组, 索引)`, `取整数值(...)`, `取对象值(数组, 索引, 结果)`, `取数组值(...)`
**置值**（静态）：`置入文本值(数组, 值, 插入索引=-1)`, `置入整数值(...)`, `置入对象值(...)`, ...
**替换**：`替换文本值(数组, 索引, 值)`, `替换路径文本值(数组, 路径, 值)`, ...
**其他**：`删除成员(数组, 索引)`, `是否存在(数组, 索引)`, `取类型(数组, 索引)`

### JSON类型（常量）

`空值`, `整数值`, `长整数值`, `小数值`, `文本值`, `逻辑值`, `数组值`, `对象值`

## w_plog

**包**：`火山.日志` | **基于plog开源日志库**

### 日志级别（常量）

`通常`, `致命`, `错误`, `警告`, `信息`, `调试`, `详细`

### 日志类（单实例，静态）

- `初始化文件(接收级别=通常, 格式对象, 文件名称="default.log", 最大长度=0, 最大数量=0)` → 日志记录器 | 文件轮替日志
- `初始化控制台(接收级别=通常, 格式对象, 输出模式=标准输出)` → 日志记录器 | 彩色控制台

### 日志输出（单实例，静态）

- `指定级别(输出级别, 输出内容...)`, `通常(...)`, `致命(...)`, `错误(...)`, `警告(...)`, `信息(...)`, `调试(...)`, `详细(...)`
- 内容支持多参数 `<<` 拼接

### 日志条件输出（单实例，静态）

同 `日志输出`，但每个方法多一个 `输出条件` 参数（逻辑型），仅条件为真时输出

### 多日志类 / 多日志输出（多实例）

同上述接口，但每个方法增加 `日志标识` 参数（整数），支持多个独立日志实例

### 日志格式预设

`日志格式预设_详细`, `日志格式预设_消息`, `日志格式预设_CSV`, `日志格式预设_功能`

### 日志输出模式（常量）

`标准输出`(行缓冲), `标准错误`(无缓冲)
