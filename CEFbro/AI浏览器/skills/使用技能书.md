# AI浏览器 — 使用技能书

> 编译输出：`linker/docs/使用技能书.md` · 在线：`http://127.0.0.1:9222/docs/使用技能书.md`

## 五分钟上手

1. 启动 **AI浏览器.exe**
2. `http://127.0.0.1:9222/health` → `status: ok`
3. Cursor 配置 `node mcp_bridge.js`（见配置说明书）
4. `browser_navigate` 打开目标 URL
5. 读操作用 sync-wait 默认（`browser_get_title`、`browser_evaluate`、`dom_*`）

```bash
node mcp_bridge.js --check
node scenarios/douyin_xhr_encrypt_scan.js --wait 20 --scroll
```

## 常用工具

- **导航**：`browser_navigate`、`browser_get_url`、`browser_reload`
- **读页面**：`browser_evaluate`、`browser_dom_query`、`browser_get_title`
- **填表**：`browser_fill_set_value`、`browser_fill_click`
- **网络**：`browser_network` list、`browser_collect` reverse_prepare
- **Hook**：`browser_inject` persist + `inject_id`
- **工作流**：`workflow_run` name=hello

POST body 不在网络层 → 用 `scenarios/douyin_xhr_encrypt_scan.js`

## Sync-Wait

默认一次返回；`async_only:true` 得 task_id；`max_ms` 调超时。单浏览器脚本**顺序**执行。
