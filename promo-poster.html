产品概述

Windows 本地 MCP 浏览器自动化服务。启动 AI浏览器.exe 后暴露标准 MCP 协议；任意 AI 代理可调用 217 个 browser_* 工具，说一句话 Agent 自动串联，无需手写 Playwright。

🎁 GitHub Release 下载即用

AI-Browser-MCP-x64-v2.6.0.zip（约 157MB）含全部 217 工具 + CEF 运行时，解压运行即可。

一句话场景

复制话术到 AI 对话 — 不用写脚本、不用记工具名。

#场景示例话术主要工具
①数据采集滚动列表，采标题价格 JSONdom_query / collect
②逆向分析扫描 POST，标疑似加密字段inject Hook / network
③定位算法断点跟到 sign 函数，给源码debugger_*
④自动填表登录后台，导出订单表fill_* / workflow
⑤工作流复用存成 JSON，下次一键跑workflow_*
📊 采集
帮我把商品列表滚动 5 屏，把标题、价格、链接采成 JSON。
🔍 逆向
打开 https://www.douyin.com ，扫描 XHR/fetch 的 POST，标出疑似加密字段并说明依据。
🎯 定位
提交表单时下断点，跟到计算 sign 的 JS 函数，把函数名和源码片段给我。
📝 填表 / RPA
登录后台，导出订单表前 20 行；把流程存成 workflow JSON。

实测：Cursor 一句话 → Hook + 滚动 → 分析 33 条 POST

3 步上手

  1. 下载并运行下载 Release zip → 解压 → 双击 AI浏览器.exe
  2. 确认服务打开 http://127.0.0.1:9222/health 看到 "status":"ok"
