3 步上手

Windows 10/11 x64 · Node.js 18+（仅 Cursor 桥接需要）· Release 已含 CEF 运行时

  1. 下载并运行 下载 AI-Browser-MCP-x64-v2.6.0.zip（~157MB）→ 解压 → 双击 AI浏览器.exe
  2. 确认服务正常 浏览器打开 http://127.0.0.1:9222/health，看到 "status":"ok"；欢迎页 127.0.0.1:9222 可复制 MCP 配置
  3. 接入 AI 代理 配置 mcp_bridge.js 连接 Cursor / Claude，或直接 HTTP POST 到 9222/mcp — 对 Agent 说任一句场景话术即可

Cursor 配置示例

{
  "mcpServers": {
    "ai-browser": {
      "command": "node",
      "args": ["路径/mcp_bridge.js"],
      "env": {
        "AI_BROWSER_MCP_HTTP_POST": "http://127.0.0.1:9222/mcp"
      }
    }
  }
}
